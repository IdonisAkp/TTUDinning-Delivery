## Project Description
TTU Dining Delivery is a web platform that allows Texas Tech students to order food from campus dining halls and have it delivered to their dorm, library, or study location. Students pay using their dining dollars account, and other students can earn money by working as deliverers with flexible schedules.
## Pages Implemented
Landing Page (index.html) - Introduction to TTU Dining Delivery with value propositions for diners and deliverers
Student Dashboard (student-dashboard.html) - Users can browse dining halls, view menus, and place orders
Company Dashboard (company-dashboard.html) - Platform admin dashboard where admins manage dining halls, menu items, and active deliveries
Project Listings (projects.html) - Available delivery jobs for student deliverers to accept jobs and earn money
